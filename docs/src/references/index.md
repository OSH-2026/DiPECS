# 参考文献 (References)

本页面汇总了 DiPECS 项目在研究与开发过程中参考的核心文献、行业标准及学术论文。

## 1. LLM 驱动的智能操作系统 (LLM-Driven / AI OS)

- **AIOS: LLM Agent Operating System**
  - *Mei, K. et al.* (2024). arXiv:2403.16971.
  - 核心贡献：定义了隔离 LLM 服务与 Agent 逻辑的内核架构。
- **OS-Copilot: Towards Generalist Computer Agents with Self-Improvement**
  - *Wu, Z. et al.* (2024). arXiv:2402.07456.
  - 核心贡献：构建了能在操作系统各要素间交互的通用型 Agent 框架。
- **UFO / UFO2: UI-Focused Agent for Windows OS Interaction**
  - *Zhang, C. et al.* (2024/2025). arXiv:2402.07939, arXiv:2504.14603.
  - 核心贡献：通过 GPT-Vision 观测 GUI 的双 Agent 架构。
- **MemGPT: Towards LLMs as Operating Systems**
  - *Packer, C. et al.* (2023). arXiv:2310.08560.
- **LLM as OS, Agents as Apps**
  - *Ge, Y. et al.* (2023). arXiv:2312.03815.
